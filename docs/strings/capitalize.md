strings/capitalize
====
Put the first letter of a string in capital

```js
function capitalize(str: string)
```

### Args

**str:string**  
The string to trim.


### Returns
The capitalized string.

### Examples

```js
import { capitalize } from '@deskpro/js-utils/dist/strings';

console.log(capitalize('abcdefg')); // Outputs Abcdefg
```
